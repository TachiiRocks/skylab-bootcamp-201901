# StayFit App

Start dev

```
$ npm start
```

Build production

```
$ npm run build
```

Add ionic android build (and open in android studio)

```
$ npm run build
$ ionic capacitor add android
$ ionic capacitor open android
```

Refresh ionic android build (and open in android studio)

```
$ npm run build
$ ionic capacitor copy android
$ ionic capacitor open android
```
