# Stayfit api

Configuration example in .env.sample

Start server
```
$ npm start
```

Start dev server
```
$ npm run dev
```

Start dev server with debug
```
$ npm run dev:debug
```

Test
```
$ npm test
```

Test with debug
```
$ npm run test:debug
```

Test one module and debug

```
$ ts-mocha <path to test file> --inspect
```

Coverage
```
$ npm run test:coverage
```
