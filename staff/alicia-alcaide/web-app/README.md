# DUCKS WORLD WEB-APP

A web-app with node, express, pug and sass.

## server

- to install dependencies

```sh
$ npm install
```

- to run server

```sh
$ node .
```

- to run server in debug mode

```sh
$ node debug .
```

- to run server in watch mode

```sh
$ nodemon .
```

## styles

- to watch and transpile styles

```sh
$ npm run scss
```
